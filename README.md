# 🏥 DocAppoint - Doctor Appointment Manager

A full-stack doctor appointment booking system built with Next.js, Node.js, Express, and MongoDB.

## 🌐 Live Site
https://doc-appoint-client-ten.vercel.app

## 🚀 Features
- Browse doctors with ratings and details  
- Book appointments using modal form  
- JWT authentication with Google/GitHub login  
- User dashboard for managing bookings  
- Update and delete appointments  
- Profile management system  
- Search doctors by name  
- Fully responsive UI  

## 🧑‍💻 Authentication
- JWT-based login system  
- Google/GitHub login (one provider)  
- Protected routes for dashboard & booking  
- Persistent login after reload  

## 🏗️ Tech Stack
Client: Next.js, React, Tailwind CSS, Axios, React Hook Form, React Hot Toast, Swiper  
Server: Node.js, Express.js, MongoDB, JWT  

## 🧠 Workflow
- Users browse doctors  
- View doctor details  
- Book appointment via modal  
- Save data to MongoDB  
- Manage bookings from dashboard  
- Update/Delete without reload  
- Update profile anytime  

## 📊 Dashboard
My Bookings:
- View, update, delete appointments  

My Profile:
- View and update user info
